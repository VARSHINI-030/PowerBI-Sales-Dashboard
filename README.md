## Project Overview

This project is an interactive Sales Performance Dashboard developed using Power BI. It helps analyze sales trends, customer behavior, regional performance, product performance, and shipping mode analysis.

## Features

* Total Sales KPI
* Total Orders KPI
* Total Customers KPI
* Monthly Sales Trend Analysis
* Sales by Region
* Sales by Category
* Top 10 Products
* Sales by State
* Sales by Ship Mode
* Interactive Slicers and Filters
* Map Visualization

## Tools Used

* Power BI
* DAX
* Microsoft Excel

## Key Insights

* Identified top-performing regions and states.
* Analyzed category-wise sales contribution.
* Tracked monthly sales trends.
* Evaluated shipping mode performance.
